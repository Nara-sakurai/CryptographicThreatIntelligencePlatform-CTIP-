# Cryptographic Threat Intelligence Platform (CTIP)

A CLI-based malware detection tool demonstrating cryptographic concepts.

## Features
1. Hash Engine (SHA-256, MD5)
2. Entropy Analysis
3. Crypto Signature Detection
4. Threat Scoring
5. VirusTotal Integration
6. Digital Signatures (RSA-PSS)

## Setup
1. Clone repository
2. `python3.13 -m venv venv`
3. `source venv/bin/activate` (or `venv\Scripts\activate` on Windows)
4. `pip install -r requirements.txt`
5. Get VirusTotal API key and add to `.env`
